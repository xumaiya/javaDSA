## üé® Design Elements Used

### Glass Morphism
```css
backdrop-blur-xl
bg-white/70 dark:bg-dark-surface/70
border-olive-light/30 dark:border-dark-border
```

### Gradient Glows
```css
bg-gradient-to-br from-olive/20 to-olive-light/20
blur-xl group-hover:blur-2xl
```

### Animations
- `animate-float` - Smooth floating motion
- `animate-float-delayed` - Delayed floating
- `animate-float-slow` - Slow floating
- `animate-gradient` - Gradient color shift
- `animate-pulse` - Pulsing effect
- `animate-bounce` - Bouncing effect

### Hover Effects
- `hover:scale-105` - Slight scale up
- `hover:-translate-y-2` - Lift effect
- `hover:blur-2xl` - Increased blur
- `group-hover:translate-x-1` - Icon movement
